Dictionary And Wordle Demo

Dan Tran, Timothy Nishiwaki

EEC 172 UC Davis Spring 2022

Project Summary:

We implemented a dictionary definition search tool and Wordle game on the CC3200 board.
Word definitions are found using a free dictionary API at https://dictionaryapi.dev/.
Multi-tap texting and controls are executed on a TV remote and received through an IR sensor.
The UI is displayed on an OLED screen through SPI.
